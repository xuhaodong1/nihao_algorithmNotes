## 单调栈

适用场景：

【单调栈】适用于找到 ```nums[i]``` 的左边 / 右边的最近一个符合某种性质的位置

伪代码：

```swift
var stack = [Int]()
for (i, num) in nums.enumerated() {
    // 找到了第一个符合条件的元素
    while !stack.isEmpty && nums[stack.last!] >= num {
        let idx = stack.removeLast()
    }
    stack.append(i)
}
```
